# SafraDigital
Aplicativo móvel em desenvolvimento para auxiliar produtores rurais no gerenciamento da safra de café, permitindo o controle de lavouras, talhões, colheitas, cálculo automático do acerto de trabalhadores
e um modelo de predição para a próxima colheita.

Este aplicativo tem como objetivo substituir processos manuais utilizados no controle da produção de café por uma solução digital simples, eficiente e acessível diretamente no campo.
A aplicação permite registrar e consultar informações da colheita, além de futuramente oferecer previsões de produção com base em dados históricos.

# Como Executar
* Clone o repositório;
* Abra o projeto no Android Studio;
* Execute em um emulador ou dispositivo físico
